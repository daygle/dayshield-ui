# DayShield UI

React + Vite frontend for DayShield management.

## Requirements

- Node.js 18+
- npm

## Quick start

```sh
npm install
npm run dev
```

By default, Vite serves on `http://localhost:5173`.

## Build

```sh
npm run build
npm run preview
```

## Notes

- The backend API is provided by `dayshield-core` (default service port `3000`).
